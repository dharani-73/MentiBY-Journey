# 🌐 HTML Practice - Day 1

This project is part of my *Full Stack Web Development learning journey* through *MentiBY* under the mentorship of *Swaroop Talks*.

## 📘 Topics Covered
- HTML paragraph tag
- Headings (h1, h2, h3)
- Clickable links
- Ordered lists

## 💻 Code Example
```html
<p>Hello World!</p>
<h1>MentiBY</h1>
<h2>Full Stack Web Development</h2>
<h3>MERN Stack</h3>
<p>The last sentence is just a paragraph.</p>
<a href="https://mentiby.com/">Visit MentiBY</a>
<ol>
  <li>MentiBY</li>
  <li>Instagram</li>
  <li>LinkedIn</li>
  <li>Google</li>
  <li>Amazon</li>
</ol>
